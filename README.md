# Quiz App

A simple, interactive quiz application built with HTML, CSS, and JavaScript. Test your knowledge across various topics with instant feedback.

## Features

- **User-Friendly Interface** - Clean and responsive design
- **Personalized Greeting** - Enter your name and get a personalized message at the end
- **Instant Feedback** - See if your answer is correct or incorrect immediately:
  - Correct answers turn **green**
  - Incorrect answers turn **red**
  - The correct answer is highlighted in green if you answer incorrectly
- **Score Tracking** - Your score is calculated and displayed at the end of the quiz
- **10 Questions** - A variety of questions covering different topics (history, geography, entertainment, gaming, etc.)
- **Easy Restart** - Restart the quiz anytime with the restart button

## How to Use

1. Open `index.html` in your web browser
2. Enter your name and click **"Start Quiz"**
3. Read each question and click the answer you think is correct
4. Get instant feedback on whether your answer was right or wrong
5. Click **"Next Question"** to continue
6. View your final score and click **"Restart"** to take the quiz again

## Files

- `index.html` - Main HTML structure
- `script.js` - Quiz logic and functionality
- `style.css` - Styling and layout
